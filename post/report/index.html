What is a report?

A report is a structured report about a conference meeting that you include orally or in writing.

Typically, the following elements should appear in the report:

  1. Target
  2. Tasks
  3. Theoretical Introduction
  4. Job Description
  5. Results
  6. Bibliography

How to make a report?

  1. A solid goal has already been set before the work is done, which is why it follows in the report. The main thing is to formulate it correctly, competently and concisely.
  2. Based on the goals, you are put forward (or you make up yourself) the tasks that must be completed in order to achieve the desired result. It is better to write tasks in the report under hotel numbers (you can also select sub-items if necessary).
  3. In the theoretical introduction, you should write information related to your work. It can be a definition of a definition along with examples, or a description of just some process/action/object.
  4. Next, you need to describe all the stages of work (it is impossible to give explanations), it is advisable to reinforce these stages with images (they need to be signed and links to them in the necessary places).
  5. The results are formulated in accordance with the objectives. They should also be short enough.
  6. The sources from where you took the information for the theoretical introduction must be indicated at the end of your report (they must be drawn up in accordance with GOST).
